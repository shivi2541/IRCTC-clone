# IRCTC Clone

This project is a railway reservation website inspired by the IRCTC platform. It is built using HTML, CSS, and JavaScript and focuses on implementing responsive web design, form validation, DOM manipulation, and local storage functionality.

## Project Overview

The website allows users to search for trains, book tickets, manage their bookings, plan journeys, and submit queries through a contact form. The project was developed as part of a web development learning task to strengthen front-end development skills.

## Features

### Train Search
- Search trains using source and destination stations.
- Display train details such as departure time, arrival time, and seat availability.

### Ticket Booking
- Book tickets for available trains.
- Generate a unique PNR number for each booking.
- Store booking information using Local Storage.

### My Bookings
- View all booked tickets.
- Cancel bookings when required.
- Automatically update booking records.

### Journey Planner
- Add destinations dynamically.
- Remove destinations from the list.
- Demonstrates JavaScript DOM manipulation.

### Contact Form
- Collect user details including name, email, phone number, and message.
- Validate user input before submission.
- Store submitted queries using Local Storage.

### Responsive Design
- Built using Flexbox and CSS Grid.
- Optimized for desktop, tablet, and mobile devices using media queries.
